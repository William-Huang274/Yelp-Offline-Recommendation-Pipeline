# Showcase History Result Surface

This directory keeps small historical result files that are still useful for
upgrade storytelling, but are no longer the active freeze baseline.

Current contents focus on:

- earlier `stage10` milestone metrics for `bucket5`
- `stage11` progression points used in the current upgrade narrative

These files are copied from the frozen local pack so that the repository can
reference them without exposing the full `_prod_runs` layout in outward-facing
documents.
